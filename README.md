# 🏘️ Egyptian Real Estate Market Dashboard

## 📌 Project Overview
This project analyzes the **Egyptian real estate market** using real listing data to extract meaningful insights about prices, locations, property features, and payment methods.
The main focus of the project is **data cleaning, rebuilding, and modeling** before visualization, ensuring the dashboard is accurate, reliable, and ready for real-world decision making.

## 🧹 Data Cleaning & Rebuilding (Key Step)
Before building the dashboard, the dataset went through a complete **data cleaning and restructuring process**, including:

- Removing duplicates and invalid records  
- Handling missing and inconsistent values  
- Standardizing city and governorate names  
- Creating derived columns (Price per Meter, Area in sqm, etc.)  
- Validating numerical fields (price, area, down payment)  
- Rebuilding the dataset into a **clean, analysis-ready model**

This step was crucial to ensure that all insights shown in the dashboard are trustworthy.

---

## 📊 Dashboard Structure
The Power BI dashboard consists of **three main pages**:

### 1️⃣ Governorate Page
- Total listings per governorate  
- Average & median prices  
- Average price per meter  
- Comparison between governorates  
- Key KPIs for each governorate  

---

### 2️⃣ City Page
- Price distribution within each city  
- Average price by property type  
- Price per meter analysis  
- Area & size insights  
- Payment method analysis  
- Time-based trends (listings & prices)  

---

### 3️⃣ Table View
- Detailed listing-level data  
- Fully filterable and searchable  
- Supports deeper exploration and validation  

---

## 🎯 Key Insights
- Clear price differences between governorates and cities  
- Strong relationship between property size and price  
- Significant variation in price per meter across locations  
- Payment methods impact pricing and accessibility
